Value Types & Modifiers

Length Units

  • 10px, 100px - Pixels
  • 1rem, 2em - Relative units
  • 50%, 100% - Percentages
  • 100vh, 50vw - Viewport units

Color Values

  • #ff0000, #f00 - Hex colors
  • rgb(255,0,0) - RGB values
  • rgba(255,0,0,0.5) - RGBA values
  • red, blue - Named colors

Modifiers

  • hover: - Hover state
  • md: - Responsive breakpoints
  • ! - Important modifier
  • md:hover: - Combined modifiers
