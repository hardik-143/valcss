# ⚡ valcss — Commands Only

Minimal cheat sheet to use valcss.

## Setup

```bash
pnpm install valcss
# or
npm install valcss
```

## Commands

```bash
# create default config
valcss init

# build once using config
valcss

# watch and rebuild on changes
valcss --watch

# override output path
valcss --output dist/styles.css

# print CSS to stdout only
valcss --dry-run

# show help
valcss --help
```
